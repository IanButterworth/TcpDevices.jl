  • set_impedance_1Mohm

Example

> scope_h = initialize(AgilentDSOX4024A)

Grab data from channel 1

> data = get_data(scope_h, 1)

Grab data from channel 2 and 4

> data_array = get_data(scope_h, [2,4])

Grab data from all available channels Plot waves from all available channels

> using Plots
> data_array = get_data(scope_h)
> plot(data_array)

Low Pass Filter Turn on Low Pass Filter 25

> lpf_on(scope_h)

Check if low pass filter is on

> get_lpf_state(scope_h) == "1"

Turn on Low Pass Filter 25MHz

> lpf_off(scope)
> get_lpf_state(scope_h) == "0"

Impedance

> set_impedance_1Mohm(scope_h)
> get_impedance(scope_h) == ONEM
> set_impedance_50ohm(scope_h)
> get_impedance(scope_h) == FIFT

Terminate TCP connection

> terminate(scope_h)

TcpInstruments.VersatilePower — Type

This device only has one channel so channel specific functions and optional arguments are not available.

Available functions

  • enable_output(obj::Instr{VersatilePower})
  • disable_output(obj::Instr{VersatilePower})
  • set_voltage(obj::Instr{VersatilePower})
  • get_voltage(obj::Instr{VersatilePower})
  • set_current_limit(obj::Instr{VersatilePower}, current)
  • get_current_limit(obj::Instr{VersatilePower})

Helpers

  • remote_mode(): sets the device to remote mode. Automatically called on initialize
  • local_mode(): sets the device to local mode. Automatically called on terminate

This instrument has a remote and local mode. Some commands do not work while the device is in local mode, thus when initializing this device remote_mode is called automatically and the device is always in remote mode by default.

These functions should not be directly needed but if for some reason you need to switch modes while using the device you can use local_mode to turn the device back to local mode.
